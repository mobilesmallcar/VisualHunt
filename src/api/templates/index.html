📖 数据预处理

作用

将原始图片转换为模型可接受的张量格式。统一尺寸、归一化像素值，为训练和推理提供标准化的输入。

处理流程

  • Resize((H, W)) — 将所有图片缩放到统一尺寸（默认 64×64）
  • ToTensor() — 将 PIL Image (H×W×C, 0-255) 转为 Tensor (C×H×W, 0-1)
  • 无 Padding — 直接 resize 填充，不保留原始宽高比

如何检查

在「预处理」页面中，使用「原生数据展示」查看原始图片，再用「预处理抽查」查看处理后的张量图。数值范围应为 [0.0, 1.0]，形状为 (3, H, W)。

📖 图像分类

作用

对时尚商品图片进行 5 分类识别：上衣、鞋、包、下身衣服、手表。

模型

轻量级 CNN（Conv + ReLU + Pool + FC），输出 5 维 logits，经 Softmax 得到概率分布。

训练

  • 损失函数：CrossEntropyLoss
  • 优化器：AdamW
  • 监控指标：Train Loss / Val Loss / Val Accuracy
  • 保存策略：验证损失最低时保存

📖 图像去噪

作用

学习从被随机高斯噪声污染的图像中恢复清晰图像。输入为噪声图，目标为原图。

模型

卷积自编码器（ConvDenoiser）：Encoder 提取特征 → Decoder 重建图像，全卷积结构保留空间信息。

训练

  • 噪声：高斯噪声，比例默认 0.5
  • 损失函数：MSELoss（像素级均方误差）
  • 优化器：Adam

📖 相似度检索

作用

将图片编码为低维嵌入向量，利用 KNN（余弦距离）在嵌入空间中检索最相似的图像。

模型

Encoder-Decoder 架构：Encoder 输出紧凑特征向量，Decoder 负责重建图像（训练用）。

流程

  • 训练 Encoder+Decoder（MSELoss 重建）
  • 为全量数据集生成嵌入矩阵 embeddings.npy
  • 查询时提取特征，KNN 找近邻

📖 原生数据说明

数据结构

项目不携带原始图像，需自行准备并按以下结构放置：

data/
├── dataset/               # 全部原始图片
│   ├── 0.jpg
│   ├── 1.jpg
│   └── ...
└── fashion-labels.csv     # 图片 id 与类别标签映射

标签文件格式

CSV 包含两列：id（图片编号，如 0, 1, 2...）和 target（类别编号 0-4）。

📖 训练与评估

训练

在「训练」页面选择任务并启动。训练在后台线程中运行，前端通过轮询实时获取损失值并绘制曲线。可随时停止。

评估

在「评估」页面选择已训练好的任务，在测试集上计算最终指标。分类任务返回 Loss + Accuracy，去噪/相似度返回 Loss。

配置影响

所有训练、评估、预处理检查均读取「配置」页面的参数。修改配置后需点击保存。
