## 📖 项目概述

**VisualHunt** 是一个融合 **图像理解、特征提取与智能搜索** 的综合型 AI 应用。灵感来源于“在海量图像中精准定位相似内容”的现实需求，适用于商品推荐、图像去重、缺陷检测、内容审核等场景。

本项目通过自编码器学习高效图像表示，结合 KNN 实现毫秒级检索，并支持图像降噪与商品分类，打造端到端的视觉智能解决方案。
